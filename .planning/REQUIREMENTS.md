# Milestone v3.0 Requirements: Social, Search & Management

## Search (SRCH)
- [ ] **SRCH-01**: User can query the Spotify API for tracks, artists, and playlists via a global search input.
- [ ] **SRCH-02**: Search results can be filtered by category (All, Artist, Playlist, Album).
- [ ] **SRCH-03**: Users can trigger playback directly from search result items.
- [ ] **SRCH-04**: Implement a "Recent Searches" section that persists locally.

## Library Management (LIB)
- [ ] **LIB-01**: User can "Like" (Save) and "Unlike" (Remove) tracks using the Heart icon, synced with their real Spotify library.
- [ ] **LIB-02**: User can create a new playlist from the sidebar.
- [ ] **LIB-03**: User can add a track to any of their own playlists via a context menu or "Add" button.

## Social & Interaction (SOC)
- [ ] **SOC-01**: Friend Activity Sidebar — Implement a right-hand sidebar for Desktop users showing simulated or real friend activity.
- [ ] **SOC-02**: Synced Lyrics — A dedicated view displaying real-time scrolling lyrics synchronized with the current playback position.
